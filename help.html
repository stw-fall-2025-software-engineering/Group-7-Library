If you’re stuck using the Smart Library app, this page gives quick answers.

Frequently Asked Questions

How do I check if a book is available?

Use the search on the main page. The book will show as Available or Checked Out. If it is checked out, you can join the waitlist.

How do I find a study space?

Open the Study Spaces section to see which areas have open seats before you walk around the library.

How do I get reminders about due dates?

When you borrow a book, the system automatically sets a due date and sends a reminder before it is due.

What if I still need help?

If this page doesn’t answer your question, please contact the library front desk or submit feedback from the Feedback page.
